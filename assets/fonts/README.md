# 字体说明

本目录下的字体均来自 [Google Fonts](https://fonts.google.com)，遵循 **SIL Open Font License 1.1**。

| 文件 | 字体 | 设计者 | 用途 |
|---|---|---|---|
| `Anton-Regular.ttf` | Anton | Vernon Adams | 超粗压缩英文，Stage 5 字幕选项 |
| `ArchivoBlack-Regular.ttf` | Archivo Black | Omnibus-Type | 饱满圆润英文，**默认字幕字体** |
| `Oswald-Bold.ttf` | Oswald | Vernon Adams | 高对比英文 |
| `Poppins-Black.ttf` | Poppins | Indian Type Foundry | 几何无衬线 |

可在 `.env` 中通过 `SUBTITLE_FONT` 切换：`archivo_black`（默认）/ `anton` / `oswald` / `poppins`。

## 许可证

OFL 1.1 全文：https://scripts.sil.org/OFL

OFL 允许：
- 自由使用、修改、再分发字体
- 用于商业项目
- 嵌入到软件、文档、网页中

OFL 禁止：
- 单独售卖字体本身
- 修改后保留原字体名（如需修改后再分发，必须改名）
